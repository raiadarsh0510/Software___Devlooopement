### **Python Lab Summary (Day 1)**

During this lab, I learned the basics of Python programming, including variables, data types, strings, lists, tuples, conditional statements, and a simple dictionary-based project.

#### **Topics Covered**

1. **Variables and Data Types**

   * Created integer, string, boolean, and complex variables.
   * Used `type()` to identify data types.
   * Worked with:

     * `int`
     * `str`
     * `bool`
     * `list`
     * `tuple`
     * `complex`

2. **String Operations**

   * Accessed characters using indexing.
   * Used positive and negative indexing.
   * Practiced string slicing.
   * Used string functions:

     * `len()`
     * `count()`
     * `lower()`
     * `upper()`

3. **List Operations**

   * Created a list containing different data types.
   * Accessed elements using indexes.
   * Retrieved the real and imaginary parts of a complex number using:

     * `.real`
     * `.imag`
   * Performed list operations:

     * `append()`
     * `insert()`
     * `pop()`
     * List reversal using `[::-1]`

4. **Tuple Operations**

   * Created tuples.
   * Used:

     * `len()`
     * `count()`
     * `index()`
   * Understood that tuples are immutable.

5. **Conditional Statements**

   * Implemented an **if-elif-else** program to calculate grades based on student marks.
   * Accepted marks using `input()`.

6. **Restaurant Menu Project**

   * Created a restaurant menu using a **dictionary**.
   * Displayed a welcome message and menu.
   * Accepted customer order and quantity.
   * Checked whether the requested item existed in the menu.
   * Calculated the total bill.
   * Generated a receipt containing:

     * Order number
     * Item name
     * Price
     * Quantity
     * Total bill

#### **Learning Outcomes**

* Understood Python syntax and basic programming concepts.
* Gained experience with string, list, tuple, and dictionary operations.
* Learned to use conditional statements for decision-making.
* Developed a simple real-world project (Restaurant Menu Billing System) using dictionaries and user input.

**Conclusion:**
This lab provided a strong foundation in Python programming. I learned how to work with different data structures, apply built-in functions, use conditional logic, and create a simple application that demonstrates practical programming concepts.
